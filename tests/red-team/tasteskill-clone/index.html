Frequently Asked.

Taste Skill is an open-source framework of SKILL.md instruction files that guide AI coding agents to produce premium, non-generic frontend interfaces. It works by banning common AI design patterns and replacing them with intentional alternatives.
Run npx skills add Leonxlnx/taste-skill in your project directory. The SKILL.md files are automatically detected by your AI agent on the next interaction.
Cursor, Claude Code, Codex, Gemini CLI, v0, Lovable, Windsurf, GitHub Copilot, and any tool that reads SKILL.md files from the project directory.
Seven variants: taste-skill (core), redesign-skill (audit), soft-skill (luxury), minimalist-skill (clean), brutalist-skill (industrial), output-skill (anti-truncation), and stitch-skill (Google Stitch integration).
Yes. Taste Skill is completely open source under the MIT license. You can use it in any project, personal or commercial.
