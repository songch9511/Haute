- Dark gray background `#1e1e1e` (NOT pure black), secondary `#242424`, border `#363636`
- Primary text `#f8f8f8` / `#dadada`, muted `#bababa`, caption `#929292`

- Centered hero with starfield-style dot background (multiple radial-gradient layers)

- Large fake editor window screenshot below hero showing: macOS window chrome (red/yellow/green dots), "Daily Notes" sidebar with vault list, markdown content area
- Sections: Nav (fixed, backdrop-blur), Hero, Features, Plugins, Sync, Publish, Testimonials, Pricing, CTA, Footer
- Soft shadows: `0px 30px 90px rgba(0,0,0,0.2)` layered
